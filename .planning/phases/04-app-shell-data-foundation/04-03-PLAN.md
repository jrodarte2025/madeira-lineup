<!-- Firestore schema from CONTEXT.md locked decisions -->
Game document shape:
```js
{
  opponent: string,           // "Springfield FC"
  date: string,              // "2026-04-05"
  status: string,            // "setup" | "1st-half" | "halftime" | "2nd-half" | "completed"
  score: { home: number, away: number },
  lineup: {
    formation: string,       // "3-3-2"
    lineups: [[], []],       // half1/half2 position arrays (snapshot at game creation)
    roster: [],              // full roster (snapshot at game creation)
  },
  events: [],                // embedded events array (NOT subcollection)
  createdAt: Timestamp,      // serverTimestamp()
}
```

Season stats document shape (seasonStats/{year}):
```js
{
  players: {
    [playerId]: {
      goals: number,
      assists: number,
      minutes: number,
      gamesPlayed: number,
      // ... other stat totals
    }
  }
}
```

- Fields: `opponent` (string), `date` (string), `status: "setup"`, `score: { home: 0, away: 0 }`, `lineup` (object with formation, lineups, roster — snapshot of current lineup state), `events: []`, `createdAt: serverTimestamp()`.

**loadGame(gameId)** — Loads a single game by ID:
- Uses `getDoc(doc(db, "games", gameId))`.
- Returns `{ id: snap.id, ...snap.data() }` if exists, `null` if not found or error.

**updateGameStatus(gameId, status)** — Updates only the status field:
- Uses `updateDoc`.
- Returns `true` on success, `false` on failure.

**updateGameScore(gameId, score)** — Updates the score object:
- Uses `updateDoc(doc(db, "games", gameId), { score })`.
- `score` is `{ home: number, away: number }`.
- Returns `true`/`false`.

**appendGameEvent(gameId, event)** — Appends an event to the events array:
- Uses `arrayUnion` for atomic append (no read-modify-write).
- Returns `true`/`false`.

**updateSeasonStats(season, playerId, statDeltas)** — Increments player stat totals:
- `season` is a year string (e.g., "2026").
- `statDeltas` is an object like `{ goals: 1, minutes: 45, gamesPlayed: 1 }`.
- Uses `setDoc` with `{ merge: true }` on `doc(db, "seasonStats", String(season))`.
- Builds dotted-path updates: `players.${playerId}.${key}` with `increment(val)` for each key/value in statDeltas.
- Returns `true`/`false`.
