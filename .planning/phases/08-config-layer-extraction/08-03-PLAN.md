Helper to use everywhere a num is rendered inside a chip/circle (ADD to src/shared/utils.js):
```javascript
// Returns the jersey display string or null if no number to show.
// Callers should render null as "don't show the number element" (no placeholder).
export function formatJerseyNum(num) {
  return (num === null || num === undefined || num === "") ? null : String(num);
}
```
